Bike Stations Service

## Installation

```bash
$ npm install
```

## Running APP

```bash
# development
$ npm run start

# update entities in dist
$ npm run build

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running Test

```bash
npm run test
```

## Run code documentation

```bash
npm run doc
```
open `localhost:8080`

## Open swagger documentation

open `https://bike-stations-snapshot.herokuapp.com/docs`
